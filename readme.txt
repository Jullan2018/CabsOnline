Instruction on how to use CabsOnline

- In the booking.html, there is a booking form will allow
the customer to fill in their information. Afterward you will receive a message with your booking reference and the pickup date/time.

- Click the link called Access Admin. In this page there are two options. One button show all the request within 2 hours and assign them to a service. Another button allows you to assign a booking to a specific customer providing the reference number in the input box.
